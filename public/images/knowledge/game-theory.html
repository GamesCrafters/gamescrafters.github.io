Intro to GamesCrafters

A student-led research and development group at UC Berkeley that explores perfect play for abstract strategy games and makes those games explorable on the web.

What Is GamesCrafters?

GamesCrafters is a long-running undergraduate research and development group at UC Berkeley. The project centers on abstract strategy games—think Tic-Tac-Toe, Connect Four, and countless original puzzles—and asks a simple but powerful question:

What happens if a computer explores every position and both players play perfectly?

🎮
Core idea: We don't just play games; we solve them. Once a game is solved, we can show players exactly which positions are winning, losing, or tied—right inside a web interface.

Students in GamesCrafters work on everything from designing new games and writing solvers, to building web frontends and infrastructure that let anyone explore those games online.

What Does It Mean to “Solve” a Game?

When we say a game is solved, we mean we know the outcome of every position assuming both players play optimally. For each position, we can answer:

  • Is this position a WIN, LOSE, or TIE for the player to move?
  • What are the best moves from here?
  • How can the opponent respond if I play optimally?

Behind the scenes, this is done by exploring the entire game tree of possible moves and working backwards from terminal positions. Later chapters will walk through game trees, position values, and solving in detail.

The Gamesman Ecosystem

GamesCrafters uses a family of tools often referred to as the Gamesman ecosystem to turn a solved game into an interactive website. At a high level:

The Pipeline

  1. Solver: You write a solver that explores the game tree and assigns values to positions.
  2. Backend: The results are stored in a backend that can answer queries like “What is the value of this position?”
  3. UWAPI (Middleware): A small service routes web requests to the correct backend and handles different versions or formats.
  4. Frontend (GamesmanUni): A web interface where players can make moves, see perfect-play hints, and explore the game.

The rest of this textbook walks through how to go from a game idea in your head to a live, deployed game on the web, built on this pipeline.

Why Students Join GamesCrafters

GamesCrafters is both a learning environment and a research project. Students are drawn to it because it blends theory, implementation, and creativity:

  • Learn recursion, search, and algorithms by implementing real solvers.
  • Work with hashing, databases, and infrastructure.
  • Design TUIs and GUIs that make perfect play approachable and fun.
  • Contribute to software that other students and players actually use.

If you like games, teaching, algorithms, or systems—and especially if you like combining them—you’re in exactly the right place.
